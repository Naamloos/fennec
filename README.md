# fennec
Matrix Client in C# / .NET MAUI

## Building
First, make sure you have the Rust toolchain installed. You can install it from [rustup.rs](https://rustup.rs/).

Then, install uniffi-bindgen-cs, which is a tool that generates C# bindings for Rust code using the UniFFI framework. You can install it using Cargo, the Rust package manager:
```bash
cargo install uniffi-bindgen-cs --git https://github.com/NordSecurity/uniffi-bindgen-cs
```
